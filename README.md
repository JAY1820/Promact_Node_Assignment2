# Promact_Node_Assignment2 User Management App

Simple app to create & view users in `users.txt`

**Features:**

- Welcome message (/)
- User creation form (/create)
- User list (/users)
- Error handling

**Start:**

1. Node.js & npm required
2. `git clone`
3. ``npm install`
4. `nodemon index.js`

**Access:** http://localhost:3000/
